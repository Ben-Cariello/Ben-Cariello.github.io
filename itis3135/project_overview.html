Overview

This webpage will be dedicated to Cast Iron Waffles. It will contain ways to help customers decide on and order exactly what they want, as well as a spot to upload a resume for those who are interested in working there.

Intended Users

This webpage is intended to be used by customers and those interested in applying.

Dynamic Functionality

All pages will have the same header and footer implemented through Javascript. The header will include a drop down navigation bar. On the home page there will be live announcements for the store. On the About page we will implement an embedded google maps location of the shop. Finally on the menu we will implement buttons to easily navigate through different areas of the menu.
